Hugo 架站教學

將 GitHub Pages 的網站改成自己的網域，並加上 https

林鼎淵

2025-01-16

如果你想要提升網站的專業感，讓它更像是自己的品牌，我會建議你購買網域，並建立 SSL 憑證。

如果你不知道如何挑選網域，可以參考筆者先前分享的文章，這篇主要分享

挑選網域

購買網域

Namecheap GoDaddy 都可以，但是 Namecheap 比較便宜

重點要看第二年之後續約的部分

購買 SSL 憑證

SSL 憑證有免費的，但免費的可能有如下問題： 1.2.3…

網域都付錢了，我建議 SSL 也直接付錢吧…因為一年真的沒多少

建立 CSR 憑證

這邊我是使用 Node.js 的 OpenSSL 來建立憑證

npm install openssl -g

https://www.namecheap.com/support/knowledgebase/article.aspx/9704/2290/generating-a-csr-on-nodejs/

然後輸入下面的指令來建立 csr & key

openssl req -nodes -newkey rsa:2048 -keyout example_com.key -out example_com.csr

過程中會問一些問題，其中「Country Name」只能填寫符合 ISO 的 2 碼，另外「Common Name」要填寫完成的域名。
