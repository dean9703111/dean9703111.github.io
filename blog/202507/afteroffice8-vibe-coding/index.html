我用 Vibe Coding 寫了一個線上提詞機

林鼎淵

2025-07-22

開始拍 YouTube 影片後，因為我總是記不住台詞，所以需要提詞機；但免費版廣告一堆，付費版又塞了一堆用不到的功能。

既然如此，我乾脆直接用 Vibe Coding 寫一款無廣告的提詞機給大家免費使用！

▋工具選擇

一開始我先用 ChatGPT 來寫程式，但隨著需求增加，他開始鬼打牆；不但新功能沒做好，連原本能用的也壞了。

就算提示詞加上各種限制，或開新的對話窗，也沒太大幫助。

所以在 Vibe Coding 的工具選擇上，我並不推薦使用 ChatGPT。

我個人更推薦使用的工具是 Cursor，他是一款內建 AI 功能的程式碼編輯器。

可以使用中文與他溝通，且每次修改都有差異比較。

不論新手學程式或老手解 bug，Cursor 都比 ChatGPT 高效。

▋使用提醒

但還是要提醒大家，AI 雖能加速開發，卻不保證一次到位。

如果你有程式的底子，那提出具體問題、給予明確方向，可以大幅縮短開發時程。

沒有程式基礎的也別擔心，先請 AI 解釋邏輯、逐步拆解，你一樣能從 0 打造自己的工具。

不過 Vibe Coding 雖然聽起來輕鬆，但背後仍需要反覆測試，以及對錯誤訊息保持耐心。

AI 是助力，不是萬能！

▋成果展示

如果你 Vibe Coding 的作品是靜態網頁，上傳到 GitHub Pages 就能向大家展示自己的成果。
